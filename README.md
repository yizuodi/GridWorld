# GridWorld
##中山大学2022软件工程中级实训 GridWorld项目

GridWorld案例提供了一个图形化环境用于可视化对象在二维网格中的交互。在这个案例中，你将设计和制造各种Actor的对象，将它们添加到一个网格中，并且根据一定的规则决定Actor的行为。
